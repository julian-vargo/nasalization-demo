# Nasalization Demo
A partial replication of Carignan et al's 2023 Arabana Study with new acoustical data.

original paper here: https://doi.org/10.16995/labphon.9152

This repository contains a parquet-format spreadsheet, code in R to process the dataset, and a handout that provides some relevant background.

Dataset: subset of Multilingual Hispanic Speech in California Corpus (Amengual, Kim, Davidson, Vargo 2025)

Acoustical extraction via Praat Mass Analyzer (Vargo 2025) https://github.com/julian-vargo/Praat-Mass-Analyzer
